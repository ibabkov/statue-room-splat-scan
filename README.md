# Philip V Statue Room Splat Scan

Photorealistic Gaussian Splat scan of a museum room with the statue of Philip V at the center and rendered with a custom focus shader.

## Main features

- Real-time Gaussian splat rendering (Spark & Three.js)
- Cinematic camera intro that glides into the exhibit
- Performance optimizations to render 150+ MB PLY
- Focus-reveal bloom with a custom Spark shader
- Mobile-friendly

## Getting Started

First, run the development server:

```bash
npm run start:dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
